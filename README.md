# SOP
### 使用Python开发的航天器轨迹预测器
#### 配置文件（config.json）：
```json
{
    "simulator":{
        "limit": 240
    },
    "planet":{
        "mass": 5.965e24,
        "radius": 6.317393e6
    },
    "spacecraft":{
        "mass": 8506,
        "init_position": [7.1104e6, 1.3e7],
        "init_velocity": [-3.5e3, 3.0e3]
    }
}
```
#### 将参数输入config.json即可
